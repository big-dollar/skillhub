## 快速入门

### 1. 注册与登录

SkillHub 支持两种登录方式：

#### GitHub 登录
1. 点击首页右上角的"登录"按钮
2. 选择"使用 GitHub 登录"
3. 授权 SkillHub 应用访问你的 GitHub 信息
4. 登录成功后自动返回首页

#### 飞书登录
1. 点击首页右上角的"登录"按钮
2. 选择"使用飞书登录"
3. 使用飞书扫描二维码或确认登录
4. 登录成功后自动返回首页

### 2. 浏览技能

- 访问首页查看所有公开技能
- 使用搜索框按关键词搜索技能
- 使用排序按钮按点赞、最新、下载排序
- 点击技能卡片查看详情

### 3. 上传技能

1. 点击"上传"按钮或访问 /upload
2. 填写技能信息：
   - 标题：技能的名称
   - 摘要：简短描述技能用途
   - 标签：添加相关标签（如 react, typescript）
   - 可见性：选择公开或私有
3. 上传包含 SKILL.md 的 ZIP 文件
4. 点击"发布技能"完成上传

## 用户功能

### 个人资料

访问 /profile 查看和管理你上传的所有技能。

在个人资料页面你可以：
- 查看所有已上传的技能
- 编辑技能信息
- 删除技能
- 分享私有技能给其他用户
- 查看技能的点赞和下载统计

### 技能可见性

#### 公开技能
- 所有人都可以查看和下载
- 会显示在首页技能列表中
- 可以被搜索到

#### 私有技能
- 只有上传者可以查看
- 被分享的用户可以查看
- 不会显示在首页
- 无法被搜索到

## 技能管理

### 编辑技能

1. 在个人资料页面找到要编辑的技能
2. 点击"编辑"按钮
3. 修改技能信息：
   - 标题
   - 摘要
   - 标签
   - 可见性
   - 重新上传 ZIP 文件（可选）
4. 点击"更新技能"保存更改

注意：编辑技能会自动增加版本号（如 V1.0.0 到 V2.0.0）

### 删除技能

1. 在个人资料页面找到要删除的技能
2. 点击"删除"按钮
3. 确认删除操作

警告：删除操作不可恢复，请谨慎操作。

### 技能详情页

在技能详情页你可以：
- 查看完整的技能信息
- 查看技能文档
- 点赞技能
- 下载技能 ZIP 文件
- 查看作者信息
- 查看标签

## 私有技能分享

### 分享技能给其他用户

1. 在个人资料页面找到要分享的私有技能
2. 点击"分享"按钮
3. 在搜索框输入用户姓名
4. 从搜索结果中选择用户
5. 点击"分享"按钮完成分享

### 管理已分享的用户

在分享对话框中你可以：
- 查看所有已分享的用户列表
- 查看分享日期
- 取消对某个用户的分享

### 被分享用户的权限

被分享的用户可以：
- 在首页看到被分享的私有技能
- 查看技能详情
- 下载技能
- 点赞技能

被分享的用户不能：
- 编辑技能
- 删除技能
- 分享给其他用户

## 管理员功能

### 成为管理员

管理员可以通过以下方式配置：

#### 方式一：环境变量
在 .env.local 中添加：
```
GITHUB_ADMIN_LOGINS=your-github-username
FEISHU_ADMIN_IDS=your-feishu-user-id
```

#### 方式二：修改配置文件
编辑 src/lib/admin/config.ts：
```typescript
export const DEFAULT_ADMIN_CONFIG: AdminConfig = {
  githubAdminIds: [],
  githubAdminLogins: ["your-github-username"],
  feishuAdminIds: ["your-feishu-user-id"],
};
```

### 访问管理后台

管理员登录后：
1. 顶部导航栏会显示"管理后台"入口
2. 点击即可进入管理后台
3. 或直接访问 /admin

### 系统概览

在管理后台首页可以查看：

#### 统计数据
- 总技能数：平台上所有技能的数量
- 公开技能：公开可见的技能数量
- 私有技能：私有技能的数量
- 注册用户：总用户数
- 总点赞数：所有技能的点赞总数
- 总下载数：所有技能的下载总数

#### 最近上传
显示最近上传的 5 个技能

#### 活跃上传者
显示上传技能最多的 5 个用户

### 技能管理

访问 /admin/skills 管理所有技能。

功能特性：
- 查看所有技能（包括私有技能）
- 搜索技能（按标题、描述、上传者搜索）
- 查看详情
- 删除技能（管理员可以删除任何技能）

### 用户管理

访问 /admin/users 查看所有注册用户。

功能特性：
- 查看所有用户（包括 GitHub 和飞书用户）
- 搜索用户（按姓名、用户名、邮箱搜索）
- 查看用户统计

## 常见问题

### 登录问题

Q: 无法登录 GitHub
A: 检查环境变量 GITHUB_CLIENT_ID 和 GITHUB_CLIENT_SECRET 是否正确配置。

Q: 无法登录飞书
A: 确保已在飞书开放平台创建应用，并正确配置了 FEISHU_APP_ID 和 FEISHU_APP_SECRET。

### 上传问题

Q: 上传失败，提示缺少 SKILL.md
A: 确保 ZIP 文件根目录包含 SKILL.md 文件。

Q: 上传后技能显示为 404
A: 如果技能是私有的，确保你是上传者或被分享者。

### 分享问题

Q: 搜索不到要分享的用户
A: 用户需要先登录过平台才会被记录到用户列表中。

Q: 被分享的用户看不到技能
A: 确保用户已登录，且分享操作成功完成。

### 管理员问题

Q: 登录后没有显示管理后台入口
A: 检查管理员配置是否正确，用户 ID 是否为字符串格式。

Q: 访问 /admin 提示没有权限
A: 确保当前登录的用户在管理员配置列表中。

### 数据存储

Q: 数据存储在哪里？
A: 所有数据存储在 data/ 目录下的 JSON 文件中。

Q: 如何备份数据？
A: 直接复制 data/ 目录即可备份所有数据。
